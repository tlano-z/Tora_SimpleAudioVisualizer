# Tora_SimpleAudioVisualizer

サムネイル画像と音声ファイルから、円形オーディオビジュアライザー付きのMP4動画を生成するローカルWebアプリです。ブラウザ上で見た目を調整し、短いプレビューを確認してからフル尺の動画を書き出せます。

## 必要環境

- Windows 10 / 11
- Python 3.10以上

## セットアップ

1. `setup_ffmpeg.bat` をダブルクリックします。
2. 終了後、`start_app.bat` をダブルクリックします。

以上で必要な準備とアプリの起動が自動で行われます。2回目以降は `start_app.bat` だけで起動できます。

## 使い方

1. サムネイル画像をアップロードします。対応形式は `jpg`、`jpeg`、`png`、`webp` です。
2. 音声ファイルをアップロードします。対応形式は `mp3`、`wav`、`m4a`、`aac` です。
3. 左側の設定パネルで動画サイズ、FPS、ビジュアライザーモード、バー本数、半径、色、背景ぼかし、脈動、切り抜き、フォントなどを調整します。
4. 画像を選択すると、右側の `静止プレビュー` タブに設定を反映したプレビューが表示されます。
5. 左側の `プレビュー` タブで開始秒と秒数を指定し、短いMP4を生成します。
6. 左側の `本番` タブで、音声全体の長さに合わせたMP4を生成します。
7. 右側の `プレビュー動画` / `本番動画` タブで動画を再生し、ダウンロードできます。
8. 設定JSONの保存・読み込みは左側の `JSON` タブで行います。

`sample/` フォルダには起動確認用の画像と音声ファイルがあります。これらの素材は [CC BY 4.0](sample/LICENSE.md) で公開しています。

## フォント

フォントを追加したい場合は、`fonts/` フォルダに `.ttf`、`.otf`、`.ttc` ファイルを入れてからアプリを再読み込みしてください。画面の `フォント` 選択欄にファイル名が表示されます。

フォントを選択しない場合は `デフォルト` が使われます。日本語テキストをきれいに表示したい場合は、日本語対応フォントを `fonts/` に配置してください。

## 中央画像の切り抜き

左側の `切抜き` タブで中央画像の切り抜き位置を指定できます。

- `切抜きX`: 正方形の中心位置を左右に移動します
- `切抜きY`: 正方形の中心位置を上下に移動します
- `切抜きサイズ`: 元画像から切り抜く正方形の大きさを変更します

画像を選択している場合は、元画像の上に白い正方形の切り抜き枠が表示されます。ここで指定した正方形範囲が、従来どおり中央の円形画像として使われます。

## ビジュアライザーモード

左側の `見た目` タブでバーの動き方を選択できます。

- `通常（低音から時計回り）`: 12時方向から時計回りに低音から高音を並べます
- `各周波数帯ごとに正規化`: 周波数帯ごとに音量差をならし、低音だけが目立ちすぎる状態を抑えます
- `円周上に分散配置`: 低音、中音、高音を円周上に分散して配置します
- `左右対称配置`: 右半分に低音から高音を並べ、左半分に同じ動きを反転配置します

## 初期値に戻す

左側の `見た目` タブにある `見た目を初期値に戻す` ボタンで、ビジュアライザーモード、バー本数、円半径、バー長、中央画像サイズ、背景ぼかし、背景暗さ、滑らかさ、バー色、グロー色、リング、脈動を初期値に戻せます。

## 出力ファイル

明示的に保存する生成物だけが `outputs/` 配下に保存されます。

- 本番動画: `outputs/equalizer_video_YYYYMMDD_HHMMSS.mp4`
- 保存設定: `outputs/settings_YYYYMMDD_HHMMSS.json`

プレビュー動画は `tmp/` 配下で一時的に生成し、画面表示用のデータを読み込んだ直後に削除します。画面上では直近のプレビューを再生・ダウンロードできますが、`outputs/` には保存されません。その他の処理中の一時ファイルも、処理完了後に削除します。

## トラブルシューティング

### FFmpegまたはffprobeが見つからない

`setup_ffmpeg.bat` をもう一度実行してください。自動ダウンロードに失敗する場合は、[Gyan.dev FFmpeg Builds](https://www.gyan.dev/ffmpeg/builds/) から `ffmpeg-release-essentials.zip` をダウンロードし、`install_ffmpeg_from_zip.bat` にドラッグ＆ドロップしてください。

### プレビュー開始位置のエラー

プレビュー開始位置が音声の長さ以上だと生成を中断します。開始秒を音声ファイルの範囲内にしてください。

### 書き出しが遅い

解像度、FPS、バー本数、FFTサイズを下げてください。プレビューでは `プレビューを軽量化` を有効にすると確認時間を短縮できます。

## クリーンアップ方法

不要になった場合は、Tora_SimpleAudioVisualizer フォルダごと削除してください。
